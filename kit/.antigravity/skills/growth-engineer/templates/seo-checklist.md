# SEO & Conversion Checklist

## 1. Meta Data
- [ ] Unique Title Tag (< 60 chars)
- [ ] Meta Description (< 160 chars)
- [ ] Canonical URL
- [ ] `og:title`, `og:description`, `og:image`

## 2. Technical SEO
- [ ] `sitemap.xml` generated
- [ ] `robots.txt` configured
- [ ] No `index` on private/dashboard pages
- [ ] Structured Data (JSON-LD) for Products/Articles

## 3. Performance (SEO Signals)
- [ ] LCP < 2.5s
- [ ] FID < 100ms
- [ ] CLS < 0.1
- [ ] Mobile-friendly (responsive)

## 4. Analytics & Conversion
- [ ] Signup event tracking
- [ ] Primary CTA click tracking
- [ ] User retention markers (e.g., "returning_user" property)
