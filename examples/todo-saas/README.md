## What Is This?

TaskFlow is a simple **Task Management SaaS** for remote teams, built from scratch using
BuildFlow Pro. It is provided as a concrete example of:

- What the planning documents look like before a single line of code is written
- How the memory layer is structured mid-project
- The quality of output BuildFlow Pro enforces

This is **not a finished app** — it demonstrates the planning phase (Phases 0–4) and early
build phase (Phase 5) of the BuildFlow Pro lifecycle.

## How to Reproduce This

1. Install BuildFlow Pro into any empty directory:

```bash
mkdir my-app && cd my-app
npx buildflow-pro@latest init
```

2. Open the project in Antigravity and run:

```
/start-production-app
```

3. Answer the 12 discovery questions. BuildFlow Pro generates all the above documents automatically.
